# 📊 Automobile Sales Dashboard

## Introduction
The Automobile Sales Dashboard is an Excel-based project designed to analyze and visualize sales performance. It transforms raw data into meaningful insights across dimensions like year, country, product, and quantity.

This dashboard helps stakeholders quickly understand trends and make data-driven decisions using interactive visuals.

---

## Objectives
- Analyze automobile sales data effectively  
- Identify yearly sales trends  
- Compare country and product performance  
- Build an interactive and user-friendly dashboard  
- Support decision-making with clear insights  

---

## Tools & Technologies
- **Microsoft Excel** – Data analysis and dashboard creation  
- **Power Query** – Data cleaning and transformation  
- **Pivot Tables & Charts** – Data summarization and visualization  

---

## Data Overview
The dataset includes:
- Year (time period)  
- Country (sales region)  
- Product (automobile category)  
- Quantity (units sold)  
- Sales value (if available)  

---

## Data Processing
- Removed missing and duplicate data  
- Standardized formats (date, text, numbers)  
- Transformed data using Power Query  
- Prepared structured data for analysis  

---

## Dashboard Features
- **Year-wise Analysis** – Tracks sales trends over time  
- **Country-wise Performance** – Compares regional sales  
- **Product Insights** – Identifies top-performing products  
- **Interactive Filters** – Slicers for dynamic analysis  
- **Visual Charts** – Bar, line, and pie charts for clarity  

---

## Benefits
- Simplifies complex data  
- Saves analysis time  
- Highlights key trends quickly  
- Improves decision-making  

---

## Conclusion
This project demonstrates how Excel can be used for effective data analysis and visualization. It showcases skills in data cleaning, dashboard creation, and business insight generation.
